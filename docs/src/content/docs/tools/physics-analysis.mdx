## Mission

Apply QM/GR analogies with explicit translation, confidence tiers, and conventional fallbacks. Home base for all 30 physics skills.

## When to Use

Use when conventional code analysis tools are insufficient. Physics-inspired metaphors are appropriate for:
- Quantum mechanics analogies: coupling, coverage, and style analysis
- General relativity analogies: technical debt, module gravitational mass, and refactoring paths

**NOT an entry point** — always arrive here from another instruction (refactor, design, review, evaluate, research, debug).

**Triggers**: "QM/GR analysis", "apply physics metaphors", "spacetime debt", "superposition of solutions", "event horizon", "arriving from refactor/design/review"
