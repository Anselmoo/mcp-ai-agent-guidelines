# GitHub-Style Animation Catalog

## Design Consistency
- **Fixed Title**: "MCP AI Agent Guidelines Server"

- **Full-width animations**: All animations span the complete 1200px width
- **Theme support**: Dark/light mode responsive
- **Accessibility**: Reduced motion support included
